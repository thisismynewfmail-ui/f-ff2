F-FPS — Texture Check

This page fetches every texture the game uses, straight from assets/textures/ and assets/sprites/, with cache: 'no-store' and a unique cache-buster — so it shows the current file on disk as your server serves it, bypassing every cache and the game's module code.

How to read it: the thumbnail is the actual served image; the big number is its real pixel size. If you replaced grass.png with a 512×512 file, its cell should show your art at 512×512. If it still shows the old 128×128 image here, then your server is not serving your edited file (wrong folder, wrong server, or a deployed/cached copy) — that is the real problem, and no game-side change can fix it. If it shows your new file here but not in game, it's the browser caching the game's JS — hard-reload the game tab (Ctrl/Cmd-Shift-R) or run it via node scripts/serve.mjs.
